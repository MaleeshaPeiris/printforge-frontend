# PrintForge Frontend

React frontend for the PrintForge 3D print shop.

## Tech Stack
- React 18 + Vite
- React Router
- Context API for auth

## Setup

1. Install dependencies:
   npm install

2. Start dev server:
   npm run dev

3. Make sure Spring Boot backend is running on port 8080

## Pages
- `/`        → Shop front
- `/orders`  → Track orders by email
- `/login`   → Admin login
- `/admin`   → Admin panel (protected)
